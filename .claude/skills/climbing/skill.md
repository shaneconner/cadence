# Climbing Skill

## Logging Convention

**Each route/attempt is logged as a separate observation.**

- 4 V-Max attempts = log `V-Max (Bouldering)` 4 times
- 3 Tension Board problems = log `Tension Board` 3 times
- 3 Redpoints = log `Redpoint (Sport)` 3 times

This preserves granularity for tracking volume over time.

---

## Locations

### Bouldering Location
**Local bouldering gym**

Typical session structure:
1. **Campus Board** (~3 exercises)
   - Up Down Pulls
   - Ladders
   - Max Ladder
2. **Tension Board** (3-4 problems)
3. **V-Max** (3-4 attempts)
4. **Endurance finisher** (pick one):
   - Boulder 4x4's
   - Extensive Endurance Climbing

### Sport Climbing Location
**Local sport climbing gym**

Typical session structure:
1. **Technique/Strength Drills** (3-6 exercises)
   - Drop Knee
   - Hover Drills
   - Flagging
   - etc.
2. **Redpoint** (3 attempts)
3. **Endurance finisher** (pick one):
   - Sport 4x4's
   - Linked Sport Routes
   - Sport route endurance drills

### Home Training
**Grip Training**
- Hangboard exercises
- No-hang device (e.g., Gripster, Tension Block)

## Quick Reference

When user says... | Log as...
------------------|----------
"did 4 V-Max" | V-Max (Bouldering) x4 (separate logs)
"3 tension board problems" | Tension Board x3 (separate logs)
"redpointed 3 routes" | Redpoint (Sport) x3 (separate logs)
"campus board session" | Ask which exercises (Up Down Pulls, Ladders, Max Ladder, etc.)
"bouldering session" | Follow bouldering location structure
"sport climbing session" | Follow sport climbing location structure
