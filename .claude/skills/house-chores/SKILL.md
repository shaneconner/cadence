# Household Chores

Household chores use light categorization - typically just one or two parent tags.

## Adding a Household Chore

```python
add_chore("Clean Kitchen",
    description="What this involves",
    parent_chores=["Household"])
```

## Common Parent Categories

- `Household` - General home maintenance
- `Cleaning` - Cleaning-specific tasks
- `Maintenance` - Repairs and upkeep
